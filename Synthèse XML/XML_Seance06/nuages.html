Les Nuages

cirrus

Ce type de nuage possède les expèces suivantes :

  • flocus
  • castellanus*
  • spissatus
  • uncinus
  • fibratus

Pas d'HydroMétéores précisées

cirrus culmine minimum 6000 mètres et maximum à 12000 mètres

cirrocumulus

Ce type de nuage possède les expèces suivantes :

  • lenticularis
  • stratiformis
  • flocus
  • castellanus*

Pas d'HydroMétéores précisées

cirrocumulus culmine minimum 5000 mètres et maximum à 7000 mètres

altocumulus

Ce type de nuage possède les expèces suivantes :

  • lenticularis
  • stratiformis
  • castellanus*
  • flocus

Pas d'HydroMétéores précisées

altocumulus culmine minimum 2000 mètres et maximum à 6000 mètres

cumulus

Ce type de nuage possède les expèces suivantes :

  • fractus
  • humilis
  • mediocris
  • congestus

Pas d'HydroMétéores précisées

cumulus culmine minimum 200 mètres et maximum à 8000 mètres

strato-cumulus

Ce type de nuage possède les expèces suivantes :

  • stratiformis
  • lenticularis
  • castellanus*

Pas d'HydroMétéores précisées

strato-cumulus culmine minimum 0 mètres et maximum à 2000 mètres

cumulonimbus

Ce type de nuage possède les expèces suivantes :

  • calvus
  • capillatus
  • mammatus

Averses (parfois de neige, de grésil ou de grêle).

cumulonimbus culmine minimum 300 mètres et maximum à 18000 mètres
