Componentes de la aplicación

El framework de aplicaciones de Android te permite crear aplicaciones interesantes e innovadoras usando un conjunto de componentes reutilizables. Esta sección explica cómo puedes generar los componentes que definen los bloques de tu aplicación y cómo conectarlos usando intents.

Artículos del blog

Uso de DialogFragments

En esta publicación mostraré cómo usar DialogFragments con la biblioteca de compatibilidad v4 (para compatibilidad con versiones anteriores en dispositivos previos a Honeycomb) para exhibir un diálogo de edición sencillo y devolver un resultado a la actividad que realiza la llamada usando una interfaz.

Fragmentos para todo

Hoy lanzamos una biblioteca estática que expone la misma Fragments API (como también el nuevo LoaderManager y algunas otras clases) de modo que las aplicaciones compatibles con Android 1.6 o posterior puedan usar fragmentos para crear interfaces de usuario compatibles con tablets.

Ejecución de múltiples subprocesos para aumentar el rendimiento

Una buena práctica cuando se crean aplicaciones sensibles es asegurarte de que el subproceso principal de tu IU realice la cantidad mínima de trabajo posible. Cualquier tarea potencialmente extensa que pueda colgar tu aplicación se debe manipular en un subproceso diferente.
